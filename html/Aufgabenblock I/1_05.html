Schreiben Sie ein weiteres my-map unter Benutzung von einer fold-Funktion (entscheiden Sie selbst welches).

Code

#lang racket

(define (my-map f lst)
  (cond
    ((empty? lst) empty)
    (else
     (foldl
      (lambda (a b) (append b (list (f a))))
      (list (f (first lst))) (rest lst)))))
